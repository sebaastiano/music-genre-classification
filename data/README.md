# Data Directory

## Dataset

This directory should contain the Spotify Tracks Dataset.

### Download Instructions

1. Go to [Kaggle - Spotify Tracks Dataset](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset)
2. Download the dataset (you may need to create a Kaggle account)
3. Extract `dataset.csv` and place it in this directory

### Expected File

- `dataset.csv` - Main dataset file containing Spotify track features

**Note**: The dataset file is not included in the repository due to its size and licensing. You must download it separately.
